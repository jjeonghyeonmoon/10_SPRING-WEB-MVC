th:if와 th:unless

- th:if: 변수 표현식을 조건식으로 사용하여 결과가 true일 때 해당 태그의 내용이 표시됩니다.
- th:unless: 조건식의 결과가 false일 때 해당 태그의 내용이 표시됩니다.

넘어온 값은 0보다 큽니다.

넘어온 값은 0보다 작습니다.

넘어온 값은 0보다 작지 않습니다.

th:each

👍th:each 는 가장 많이 사용하는 태그로서 제일 중요!👍

이름 나이 성별 주소

th:each 는 두 번째 매개변수 stat 을 추가하게 되면, 반복 상태를 확인할 수 있다.

    index : 0 부터 시작하는 인덱스 값
    count : 1 부터 시작하는 값
    current : 현재 객체의 정보
    even : 짝수 번째 여부
    odd : 홀수 번째 여부
    first : 첫 번째 데이터 여부
    last : 마지막 데이터 여부
이름 나이 성별 주소 index count size current even odd first last
