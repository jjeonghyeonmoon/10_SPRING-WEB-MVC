Thymeleaf 조건문

th:if와 th:unless

- th:if: 변수 표현식을 조건식으로 사용하여 결과가 true일 때 해당 태그의 내용이 표시됩니다.
- th:unless: 조건식의 결과가 false일 때 해당 태그의 내용이 표시됩니다.

넘어온 값은 0보다 큽니다.

넘어온 값은 0보다 작습니다.

넘어온 값은 0보다 작지 않습니다.
